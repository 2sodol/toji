# 지도에 이미지 레이어 띄우기 정리

## 1. 라이브러리

- **OpenLayers 3** (v3.20.1)
- **사용 클래스:**
  - `ol.source.ImageStatic` - 정적 이미지 소스
  - `ol.layer.Image` - 이미지 레이어

---

## 2. 주요 속성

### ImageStatic Source

- `url`: 이미지 파일 경로
- `imageExtent`: 이미지 표시 영역 `[minX, minY, maxX, maxY]` (EPSG:3857)
- `projection`: 좌표계 (`"EPSG:3857"`)
- `crossOrigin`: CORS 설정 (`"anonymous"`)

### Image Layer

- `source`: ImageStatic 소스 객체
- `zIndex`: 레이어 순서 (6)
- `opacity`: 투명도 (0.8)

---

## 3. 사용 방법

```javascript
// 1. 이미지 표시 영역 계산 (중심점 기준 ±29m)
var extent = [centerX - 29, centerY - 29, centerX + 29, centerY + 29];

// 2. 소스 생성
var source = new ol.source.ImageStatic({
  url: "/image.png",
  imageExtent: extent,
  projection: "EPSG:3857",
  crossOrigin: "anonymous",
});

// 3. 레이어 생성
var layer = new ol.layer.Image({
  source: source,
  zIndex: 6,
  opacity: 0.8,
});

// 4. 지도에 추가
map.addLayer(layer);
```

---

## 4. 이미지 표시 영역

- 중심점 좌표 기준 ±29m (총 58m × 58m)
- 좌표계: EPSG:3857 (Web Mercator)

---

## 5. 주요 함수

- `updateImageLayer(centerX, centerY, imagePath)`: 이미지 레이어 추가
- `toggleImageLayer(show)`: 이미지 레이어 표시/숨김
- `clearImageLayer()`: 모든 이미지 레이어 제거
